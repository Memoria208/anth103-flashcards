## Flashcard App (`ANTH103_Flashcards.html`)

A standalone HTML file — no server or install required. Open directly in any browser.

**Features:**
- Chapter selection grid (Ch. 1–12; 1–3 populated)
- Flip animation with front/back card reversibility
- Tag filter chips for targeted review
- "Know it / Still learning" ratings with persistent progress tracking
- Stats and progress bars per chapter
- Completion screen

**To use:** Open `ANTH103_Flashcards.html` in a browser. No internet connection required.

## Course Structure

7 modules over 8 weeks. Deadlines fall on **Tuesdays, Thursdays, and Sundays** at 11:59 PM Mountain Time.

**Assignment types:** Readings, videos, journals, timed quizzes, discussion boards, article review, research paper (scaffolded across modules).

**Research Paper milestones:**
- Abstract — Module 02
- Introduction — Module 03
- Draft (peer review) — Module 04
- Final paper — Module 06 (July 12, 100 pts)

**Textbook:** *Archaeology*, 7th Ed. — Kelly & Thomas (Cengage, 2017). ISBN: 9781305670402
